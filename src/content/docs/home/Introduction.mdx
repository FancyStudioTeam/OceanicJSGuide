Welcome to the Oceanic.js guide. In this guide we will show you how to make a bot using the `oceanic.js` dependency and we will explain you about some Oceanic.js topics.

:::note
Before starting with the guide, we think that you already have a basic knowledge about programming and JavaScript, so that you can understand the references and explanations in the guide.

If you need help, you can join the [Discord Server](https://discord.gg/xZ4AhdYrf9) and let us know about your doubts or questions.
:::

## Prerequisites

- Basic knowledge of programming and JavaScript
- Node.js version `18.13.0` or `higher`
- A code editor (Recommended [Visual Studio Code](https://code.visualstudio.com/))
