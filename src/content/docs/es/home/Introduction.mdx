:::note
Esta guía está diseñada para usuarios de Windows y Node.js.
:::

Bienvenido a la guía de Oceanic.js. En esta guía te mostraremos cómo hacer un bot utilizando la dependencia <a href="https://oceanic.ws/" target="_blank">oceanic.js</a> y te explicaremos algunos temas relacionados con Oceanic.js.

:::note
Antes de comenzar con la guía, creemos que ya tienes un conocimiento básico sobre programación y JavaScript, para que puedas entender las referencias y explicaciones en la guía.

Si necesitas ayuda, puedes unirte al <a href="https://discord.gg/xZ4AhdYrf9" target="_blank">Servidor de Discord</a> y hacernos saber tus dudas o preguntas.
:::

## Prerrequisitos

- Conocimientos básicos de programación y JavaScript
- Versión de Node.js `18.13.0` o superior
- Un editor de código (Recomendado <a href="https://code.visualstudio.com/" target="_blank">Visual Studio Code</a>)
